# ChinaCity-DataMap 中国城市人均数据地图

## 1. 项目概述

### 1.1 项目背景

随着中国经济的快速发展，各省市的人均数据差异日益显著。GDP、居民收入、房价、人口等数据是衡量地区发展水平的重要指标。本项目旨在提供一个直观、交互式的可视化平台，展示中国各地区（精确到地级市）的人均数据对比。

### 1.2 项目目标

- 展示中国地级市及以上行政区的真实人均统计数据
- 提供多种图表形式（地图、柱状图、折线图、散点图等）
- 支持多维度数据切换（GDP、收入、房价、人口等）
- 数据定期更新，保证时效性

## 3. 数据设计

### 3.1 数据维度

| 指标 | 单位 | 数据来源 | 更新频率 |
|------|------|----------|----------|
| 人均GDP | 万元 | 国家统计局 | 年度 |
| 人均可支配收入 | 元 | 国家统计局 | 年度 |
| 商品房均价 | 元/㎡ | 国家统计局/住建部 | 季度 |
| 常住人口 | 万人 | 国家统计局 | 年度 |
| GDP总量 | 亿元 | 国家统计局 | 年度 |

### 3.2 数据库模型

#### 3.2.1 城市基础表 (cities)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| name | String(50) | 城市名称 |
| province | String(50) | 所属省份 |
| lat | Float | 纬度 |
| lng | Float | 经度 |
| level | Integer | 行政级别(1省2市3县) |
| city_code | String(20) | 城市代码 |

#### 3.2.2 统计数据表 (city_stats)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| city_id | Integer | 城市ID(外键) |
| year | Integer | 数据年份 |
| per_gdp | Float | 人均GDP |
| per_income | Float | 人均收入 |
| house_price | Float | 房价 |
| population | Float | 常住人口 |
| gdp_total | Float | GDP总量 |

### 3.3 数据采集策略

1. **国家统计局** (stats.gov.cn) - 主要数据源
2. **地方统计局** - 补充数据
3. **第三方数据** - 房价数据（国家统计局+计算）

## 4. API设计

### 4.1 接口列表

#### 获取所有城市
```
GET /api/cities
Response: {
  "cities": [
    {"id": 1, "name": "北京市", "province": "北京市", "lat": 39.9, "lng": 116.4}
  ]
}
```

#### 获取城市列表（含数据）
```
GET /api/cities/with-stats?year=2023
Response: {
  "cities": [
    {
      "id": 1, "name": "北京市", "province": "北京市",
      "lat": 39.9, "lng": 116.4,
      "stats": {"per_gdp": 19.0, "per_income": 75002, "house_price": 35000}
    }
  ]
}
```

#### 获取省份排名
```
GET /api/ranking/province?metric=per_gdp&year=2023
Response: {
  "ranking": [
    {"rank": 1, "province": "北京市", "value": 19.0}
  ]
}
```

#### 获取城市排名
```
GET /api/ranking/city?metric=per_gdp&year=2023&limit=50
Response: {
  "ranking": [
    {"rank": 1, "city": "北京市", "value": 19.0, "province": "北京市"}
  ]
}
```

#### 获取历年趋势
```
GET /api/trend/{city_id}?metrics=per_gdp,per_income
Response: {
  "city": "北京市",
  "trend": {
    "years": [2020, 2021, 2022, 2023],
    "per_gdp": [16.4, 17.0, 18.0, 19.0],
    "per_income": [65000, 68000, 72000, 75002]
  }
}
```

```

│                                                          │

└──────────────────────────────────────────────────────────┘
```

### 5.2 功能模块

#### 5.2.1 中国地图 (ChinaMap)
- 支持省份下钻到城市
- 热力图模式：按数据值染色
- 散点模式：圆点大小表示数值
- 悬停提示：显示城市详情
- 点击选中：锁定城市，图表联动

#### 5.2.2 指标选择
- 人均GDP
- 人均可支配收入
- 商品房均价
- 常住人口
- GDP总量

#### 5.2.3 年份选择
- 支持选择 2020-2024 年数据

#### 5.2.4 图表组件

| 图表 | 功能 |
|------|------|
| 柱状图 | 展示各省份/城市排名TOP10 |
| 折线图 | 展示单个城市历年数据变化 |
| 散点图 | 展示两个指标的关联（如收入vs房价） |
| 饼图 | 展示人口/经济占比分布 |

### 5.3 交互设计

- 地图与图表联动：选中城市时，图表高亮对应数据
- 图表间联动：点击柱状图某条，地图定位到对应城市
- 响应式布局：支持不同屏幕尺寸

## 9. 数据获取说明

### 9.1 真实数据源

1. **国家统计局** (http://www.stats.gov.cn/)
   - 年度数据：GDP、人口、收入
   - 数据较大，需分年获取

2. **各省统计局**
   - 补充地方数据

3. **计算获取**
   - 人均GDP = GDP总量 / 人口
   - 房价数据需从多个来源整合

### 9.2 初始数据策略

由于国家统计局数据获取需要复杂流程，第一阶段采用：
- 部分真实数据（国家统计局公开数据整理）
- 补充基于公开报告的合理估算数据
- 建立数据更新机制，后续可定期同步
